# Mode KB — POSITIONING (Brand & Positioning)

## Purpose
Nail the crispest possible answer to "what is this, who is it for, and why
does it matter — in one line?" Build the positioning layer cake that
everything else (marketing, sales, design) hangs off of.

## Inputs needed

- Product / service at one-sentence depth
- Primary customer segment (ideally with a specific job-to-be-done)
- 2–3 known alternatives the customer might otherwise use
- Any existing brand tone or company DNA signals

If any are missing, ask one question.

## Output — the positioning stack

### 1. Category frame
*What category does this compete in — and should it?*
- Category choice (existing crowded / adjacent / create new)
- Trade-offs of each choice (one line each)
- Recommendation with a one-line reason

### 2. Positioning statement (April Dunford structure)
```
For {target segment}
Who {context / job-to-be-done}
{Product} is a {category}
That {unique value}
Unlike {current alternative}
We {key differentiator}
```

Produce 3 variants at different aggressiveness levels:
- **Safe** (broad appeal)
- **Sharp** (excludes wrong-fit customers, delights right-fit)
- **Contrarian** (polarizing, strong fan/hater split)

Recommend one with reasoning.

### 3. Tagline options — 6 candidates
Structure the 6 as a 2×3 grid:

|  | Literal | Evocative |
|---|---|---|
| Short (≤4 words) | ... | ... |
| Medium (5–8) | ... | ... |
| Assertive | ... | ... |

Mark the one recommendation. Explain which customer psychology each pulls.

### 4. Voice & tone rubric
3 adjectives that ARE the brand + 3 adjectives that AREN'T.
Then 3 do/don't pairs with concrete phrasing examples:

```
DO say:  "Revenue doubled in Q3."
DON'T:   "We saw a strong performance trajectory in the third quarter."
```

### 5. Messaging pillars — 3 to 5
For each pillar: headline (5–8 words) + one-paragraph expansion + proof
point (data, quote, or demonstrable fact).

### 6. Competitive differentiation matrix

| Attribute | Us | Competitor A | Competitor B | Competitor C |
|---|---|---|---|---|

Pick 5–8 attributes that MATTER (not features they don't care about).
Mark cells ✔ / ✖ / ~. End with 2–3 sentences on where we credibly lead
and where we're credibly equal-or-behind (admitting truth is more
persuasive than everything-checkmark).

### 7. Name / identity starters (if new brand)
- 6 name candidates with one-line rationale
- Cross-check: .com availability signal, pronunciation, translation
  landmines

## Brutal honesty guardrails

- If the positioning fundamentally doesn't work (solving for non-existent
  problem, indistinguishable from competitors), say so in one paragraph
  before producing the stack.
- Don't manufacture differentiation that isn't real. "First" and "best"
  claims must be defensible.
